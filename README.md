# PauliONE Website

Dies ist der offizielle Quellcode für die PauliONE Web-Agentur.

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- Vanilla CSS für komplexe Animationen

## Lokale Entwicklung
Um das Projekt lokal zu starten:
```bash
npm install
npm run dev
```

Der Server läuft unter [http://localhost:3002](http://localhost:3002).
